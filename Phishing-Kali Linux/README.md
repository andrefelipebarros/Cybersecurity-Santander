
# Phishing para Captura de Senhas do Facebook

## Ferramentas

- **Kali Linux**
- **Setoolkit**

## Configurando o Phishing no Kali Linux

1. Acesse o modo root: 
   - Acesso root: ``` sudo su ```
2. Inicie o setoolkit:
   - Iniciando o setoolkit: ``` setoolkit ```
3. Selecione o tipo de ataque:
   - Tipo de ataque: ``` ➤Social-Engineering Attacks ```
4. Escolha o vetor de ataque:
   - Vetor de ataque: ``` ➤Web Site Attack Vectors ```
6. Selecione o método de ataque:
   - Método de ataque: ``` ➤Credential Harvester Attack Method ```
7. Escolha o método de ataque específico:
   - Método de ataque: ``` ➤Site Cloner ```
8. Obtenha o endereço IP da sua máquina:
   - Obtendo o endereço da máquina: ``` ifconfig ```
9. Informe a URL para o clone:
   - URL para clone: http://www.facebook.com
## Resultado

- **Vizualizar E-Mail**
- **Vizualizar Senha**
